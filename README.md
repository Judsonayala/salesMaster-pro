# SalesMaster Pro 🚀 

**SalesMaster Pro** es una herramienta interactiva desarrollada en Python para automatizar el cálculo de comisiones y bonos de ventas. Este proyecto demuestra el uso de estructuras de control, validación de datos y limpieza de interfaz.

## 📋 Características
- **Cálculo Dinámico:** Comisión basada en una tasa del 10%.
- **Sistema de Bonificaciones:** Bono automático de $30.000 al alcanzar la meta de $100.000.
- **Interfaz Limpia:** Uso de comandos de sistema para limpiar la terminal en cada ciclo.
- **Manejo de Errores:** Bloques `try-except` para prevenir fallos por entradas no numéricas.
- **Bucle de Usuario:** Procesamiento de múltiples vendedores mediante un ciclo `while`.

## 🛠️ Tecnologías
- Python 3.x
- Visual Studio Code
- Módulo `os`
